🦆 Ducklist - Gamified Productivity App
Ducklist adalah aplikasi to-do list produktivitas yang dirancang untuk membuat rutinitas harianmu lebih menyenangkan. Ditemani oleh teman bebek virtual, kamu bisa menyelesaikan misi, fokus dengan Pomodoro, dan mengumpulkan koin untuk mendandani bebekmu!

✨ Fitur Unggulan
Duck Companion: Teman bebek virtual yang bereaksi terhadap produktivitasmu.

Smart To-Do List: CRUD tugas lengkap dengan kategori, prioritas, dan filter berdasarkan deadline.

Pomodoro Focus: Sesi fokus terintegrasi untuk menjaga konsentrasi.

Gamification System: Dapatkan koin setiap kali menyelesaikan tugas dan capai target harianmu.

Progress Tracking: Pantau persentase penyelesaian tugas harianmu secara real-time.

Dual Theme: Mendukung Light Mode yang ceria dan Dark Mode yang menenangkan mata.

🛠️ Tech Stack & Arsitektur
Frontend: Flutter (Mobile Development)

Backend: Firebase (Authentication & Cloud Firestore)

Networking: Dio (untuk integrasi API yang handal)

State Management: Provider (Manajemen state yang efisien dan modular)

Architecture: Mengikuti prinsip modularisasi untuk memastikan kode rapi dan mudah dikembangkan.

🚀 Cara Menjalankan Aplikasi
Persyaratan
Flutter SDK (Versi terbaru)

Konfigurasi Firebase (file firebase_options.dart)

Android Studio / VS Code

Langkah-langkah
Clone Repository:

Bash
git clone [link-repo-kamu]
cd ducklist
Install Dependensi:

Bash
flutter pub get
Konfigurasi Backend:
Pastikan file firebase_options.dart sudah terpasang. Simpan konfigurasi API (jika ada) dalam file .env.

Jalankan Aplikasi:

Bash
flutter run
